image editor tutorial

the image editor allows you to create frames for your movie.
when opening the image editor frame, you are given three choices :

  • creating a new blank image
  • opening an existing image on your computer
  • opening a (or multiple) existing project's frame(s).

image editor workspace

after opening an image, you will be presented with a new menu item "editor"
from which you can launch new small frames that allow you to :

  • choose a different tool from the tool's list
  • choose a drawing color
  • change your current tool's settings
